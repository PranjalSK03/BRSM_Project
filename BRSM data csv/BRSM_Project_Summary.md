# BRSM Project 2026 -- Movie Memory Experiment Summary

## 📌 Overview

This project is based on **Event Segmentation Theory (EST)** (Zacks et
al., 2007).\
The core idea is that humans naturally break continuous experiences
(like watching a video) into meaningful **event boundaries**. These
boundaries occur when prediction error increases (e.g., change in
location, action, camera cut).

Event boundaries are believed to enhance memory encoding --- this is
known as the **Boundary Advantage Effect**.

------------------------------------------------------------------------

## 🧠 Key Theoretical Background

### 1️⃣ Event Segmentation Theory (Zacks, 2007)

-   The brain constantly predicts what will happen next.
-   When something unexpected happens → prediction error increases.
-   The brain updates its internal "event model."
-   These update moments become event boundaries.
-   Memory encoding is stronger at these boundaries.

### 2️⃣ Boundary Advantage (Swallow et al., 2009)

-   Objects presented at event boundaries are remembered better.
-   Boundary moments act as anchors in long-term memory.

### 3️⃣ Disruption Effect (Boltz, 1992)

-   Interruptions placed at natural boundaries improve memory.
-   Interruptions placed mid-event impair memory.

------------------------------------------------------------------------

## 🎬 Current Experiment Design

### Phase 1 -- Boundary Annotation

-   40 YouTube Shorts selected.
-   Independent annotators marked coarse event boundaries.
-   Consensus boundaries computed.

### Phase 2 -- Two Video Conditions

#### 🟢 Natural Cut (NB)

-   Videos end naturally.
-   Event boundaries preserved.

#### 🔴 Abrupt Cut (AB)

-   Videos interrupted 1--5 seconds *before* natural boundary.
-   Resume at onset of next event context.
-   This disrupts event model updating.

------------------------------------------------------------------------

## 👥 Participants

Two independent groups: - NB group (Natural videos) - AB group (Abrupt
videos)

------------------------------------------------------------------------

## 🧪 Recognition Task

After watching videos: - Two frames shown (1 target, 1 lure). -
Participants select previously seen frame. - Confidence rating (1--5). -
Response time recorded.

Frame types: - BB → Before boundary frames - EM → Event-middle frames

------------------------------------------------------------------------

### 2️⃣ Response Time (RT)

-   Is NB faster than AB?

### 3️⃣ Confidence Ratings

-   Does boundary manipulation affect certainty?

### 4️⃣ Derived Indices

-   Recognition Memory Index (REC)
-   Lure Discrimination Index (LDI)

### 5️⃣ Vigilance Exclusion

-   Participants with encoding time \> 27.05 minutes excluded.

------------------------------------------------------------------------

## 🎯 Core Research Question

Does artificially disrupting natural event boundaries reduce the
boundary-related memory advantage predicted by Event Segmentation
Theory?

------------------------------------------------------------------------

## 📌 Expected Pattern (If Theory Holds)

-   NB group → higher recognition accuracy.
-   Stronger advantage for BB frames in NB group.
-   AB group → reduced boundary advantage.
-   EM frames → minimal difference between groups.
